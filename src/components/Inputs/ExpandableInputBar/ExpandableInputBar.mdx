# ExpandableInputBar

> A simple customizable and expandable input bar to do magical things.

<CodeBlock title="Default">
  <ExpandableInputBar
    buttonText="Find User"
    placeholder="Search users"
    expandedButtonText="Find them!"
    onClick={value => {
      alert("Search term: " + value);
    }}
  />
</CodeBlock>
