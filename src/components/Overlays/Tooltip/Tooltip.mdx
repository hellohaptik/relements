# Tooltip

> A clickable UI element with active and hover states. Indicates to the user that it's clickable.

<CodeBlock>
  {(value, setValue, ref) => (
    <div>
      <Button innerRef={ref} onClick={() => setValue(true)}>
        Open Tooltip
      </Button>
      <Tooltip
        attachTo={ref}
        onClose={() => setValue(false)}
        active={value}
        offset={{ top: 8 }}
      >
        <div style={{ padding: 8, background: "#FFF" }}>This is a Tooltip</div>
      </Tooltip>
    </div>
  )}
</CodeBlock>
