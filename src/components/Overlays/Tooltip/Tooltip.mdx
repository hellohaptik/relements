# Tooltip

> A clickable UI element with active and hover states. Indicates to the user that it's clickable.
> Adheres to Haptik Design System.

<CodeBlock>
  {(value, setValue, ref) => (
    <>
      <Button
        themed
        variant="secondary"
        innerRef={ref}
        onClick={() => setValue(true)}
      >
        Open Tooltip
      </Button>
      <Tooltip
        attachTo={ref}
        themed
        onClose={() => setValue(false)}
        active={value}
        offset={{ top: 8 }}
      >
        <div>This is a Tooltip</div>
      </Tooltip>
    </>
  )}
</CodeBlock>
